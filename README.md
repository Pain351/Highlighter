# Highlighter
A chrome extension that collects the notes from different sites with the source link which preserve the data even after closing the browser. As well as one can download a Markdown file from it.

## Description:
* It is the chrome extension which let's you collect the data from the different websites rather then a page and let you download those data in a Markdown file. It doesn't break the flow of the research to collect data as well as the markdown file has the source from where the note was taken so you can always go through the content.
